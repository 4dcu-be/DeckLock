Description

Schirrú list I played in Pro Rank during the Season of the Draconid in 2020. This is the version with Saesenthessis which is less draw dependant as the Sheldon Skaggs version. This version also allows you to get some no-unit value the last round which can put Blaze of Glory decks in a really strange position.

Typically you'll want to win R1 if possible, bleed round two and finish with Harald Gord. Though in case R1 is lost, you can fight the bleed and hopefully take the last round using Harald Gord and carry over.
