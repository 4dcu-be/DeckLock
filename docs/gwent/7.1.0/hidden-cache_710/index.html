Description

One of the decks I played on Pro Ladder during the Season of the Draconid in 2020. With Hidden Cache gaining a boost this deck became viable again. Getting round control is important so usually some engines need to come down, in round two Passiflora can be used to bleed the opponent while keeping something to take the final round and the game.
