Description

After spending some time trying to get Ciri to work, which didn't happen, I landed on this list to place and climb a little on Pro Rank during the Season of the Draconid 2020. It is the old double ball list, upgraded with some new cards for consistency. Since this is not a devotion deck, Usurper will not get the aristocrat tag the last round, so use aristocrats with care.
