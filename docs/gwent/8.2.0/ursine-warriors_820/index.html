Description

Warriors list with Cerys, Blueboy and Ursine ritual along with a few control tech cards like Gremist and Djenge Frett. Traditionally I’ve been doing well with SK Warriors in various shapes and the amount of control combined with putting some bodies on the board is a strategy I like in other card games too. If SK Warriors is viable it is an obvious choice for me to include.
